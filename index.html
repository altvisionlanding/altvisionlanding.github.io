Altvision

Auto-generated alternate tags.

Generate

Utilizes Microsoft Open CV platform to analyze the image and create an auto-generated description.

Input

Inputs the description produced into the site's html as an alternate tag.

Read

The site is now compatible with screen readers who will read out the caption.

In early March 2018, I had the opportunity to act upon my concerns at a hackathon I attended, CodeDay Washington. There I worked on a chrome extension that uses artificial intelligence through computer vision to go from image-to-text, inserting an auto generated “alt tag” into a site’s HTML to ensure compatibility with screen readers. Alt tags, an essential feature for the visually impaired however is unfortunately not implemented in many websites and application. My solution was to create auto-generated alt tags that would be inserted into a web page the user is visiting so ensure compatibility with screen readers. I worked on the application in Python and Javascript and won the Special Award for Accessibility and AI.
